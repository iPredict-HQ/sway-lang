## How to Collect Feedback

1. Share the deployed testnet URL with testers
2. Ask them to connect their wallet and perform these actions:
   - Browse and filter markets
   - Place at least one bet (YES or NO)
   - Check the leaderboard
   - View their profile page
   - Try the referral system (register + share link)
   - Claim rewards on a resolved market
3. Record their experience using the template above
4. Prioritize and address high-severity feedback first

## Feedback Categories

- **UX / UI** — Layout issues, confusing flows, mobile responsiveness
- **Performance** — Slow loading, transaction timeouts
- **Functionality** — Features not working as expected
- **Content** — Unclear labels, missing information
- **Accessibility** — Keyboard navigation, screen reader support
